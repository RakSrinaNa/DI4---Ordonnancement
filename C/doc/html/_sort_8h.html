Duplicates a solution then moves a delivery backwards.

Parameters
solutionThe solution. Not null.
taskThe task to move.
shiftThe number of packs away the task shall be moved backwards to.
Returns
The duplicated and moved solution.
Remarks
The user is in charge of freeing the returned value.
If the task doesn't exist, no move is performed but the solution is still duplicated.
If shift implies that the task is moved outside of the number of packs, it is moved to the first pack.
